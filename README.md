


## BOOK-MOVIES-APP
Search and Book movies from OMDB movies api.

## Running the Applicaction Locally.

```sh
git clone https://github.com/Awesome94/BookMoviesApp.git or Fork Repo
```
```sh
cd into the project
```

```sh
npm install
```

```sh
npm start
```

If you wish to connect to the backend, NODE API can be found here: https://github.com/Awesome94/BookMoviesApi
